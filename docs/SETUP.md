# GitHub Pages Setup

## GitHub Pages aktivieren

1. Oeffne dein Repository auf GitHub.
2. Gehe zu `Settings`.
3. Klicke links auf `Pages`.
4. Stelle `Build and deployment` auf `Deploy from a branch`.
5. Waehle den Branch `main`.
6. Waehle den Ordner `/docs`.
7. Speichere die Einstellung.

Nach kurzer Zeit ist die Seite unter deiner GitHub-Pages-URL erreichbar.

## Bereits vorbereitet

- `index.html` als Startseite
- `styles.css` fuer das Layout
- `social-preview.svg` als Social-Preview-Asset
- Meta-Tags fuer Description, Open Graph und Twitter
