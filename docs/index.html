Warum dieses Projekt?

Nicht nur Code, sondern ein echter Lernpfad

Verstaendlich aufgebaut

Jedes Thema hat Dokumentation, Beispielcode, Aufgaben und Loesungen.

Anfaengerfreundlich

Die Inhalte sind bewusst so geschrieben, dass man sie ohne Vorwissen Schritt fuer Schritt durcharbeiten kann.

Zum Wiederholen geeignet

Das Projekt funktioniert nicht nur zum Lernen, sondern auch als spaeteres Nachschlagewerk.

Zielgruppe

Fuer wen dieses Repository gedacht ist

  • Einsteiger, die C++ systematisch lernen wollen
  • Studierende und Schueler
  • Menschen, die Pointer, Referenzen und OOP endlich sauber verstehen wollen
  • Alle, die eine deutschsprachige Lernsammlung auf GitHub suchen
